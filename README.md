# Scholars Bookmarklets & Tools

A small collection of JavaScript bookmarklets and snippets I made for fun and testing on iOS devices (mostly Safari on iPad).

Nothing fancy — just some quick experiments with overlays, iframes, and basic page manipulation.


All of them are just single-file JS. No dependencies, no build step.

## How to use

1. Open the file you want  
2. Copy the entire code  
3. In Safari: Add a new bookmark → paste into the URL field (starts with `javascript:`)  
4. Save and tap it on any page to run

Tested on iPadOS 18 and iOS 18. Works best with JavaScript enabled in Settings > Safari.

## Notes

- Bookmarklets can be finicky on supervised devices (school iPads) — if nothing happens, check JavaScript is on  
- The iframe thing sometimes gets blocked by sites like YouTube (X-Frame-Options header) — normal, not a bug  
- These are for learning/playing around only — don't break rules or anything  

If you use one and it breaks, lmk what happens.
